Dans ce dossier vous trouverez le code HTML pouvant prendre une photo en 
utilisant le webRTC qui est une API javascript.

L'API ne sert pas seulement à prendre des photos mais à manipuler tous 
type de flux média (Possibilité de faire des visio conférence etc...).
